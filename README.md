# VoiceCAD

VoiceCAD is an AI-powered voice-controlled CAD assistant built using Python, Whisper AI, and Fusion 360 API.

## Features
- Voice-to-text conversion using Whisper AI
- Command parsing using NLP
- Automatic 3D CAD model generation
- Fusion 360 integration

## Tech Stack
- Python
- Whisper AI
- Fusion 360 API
- FFmpeg

## Workflow
Voice Input → Speech Recognition → Command Parsing → Fusion 360 API → 3D Model
